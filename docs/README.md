# Evolution Simulator (진화 시뮬레이터)

> AI 기반 생태계 진화 시뮬레이션 게임

## 프로젝트 개요

빈 원시 세계에서 시작하여 생명이 스스로 탄생하고 진화하는 과정을 관찰하는 웹 브라우저 기반 시뮬레이션 게임입니다. 사용자는 창조자가 아닌 **관찰자이자 재앙의 신**으로서, 생태계에 재앙만 가할 수 있으며, 생명체들은 AI를 통해 스스로 진화하고 적응합니다.

## 핵심 컨셉

### 🎯 게임 목표
- 빈 세계에서 생명의 탄생과 진화를 관찰
- 다양한 재앙을 통해 진화의 방향에 영향을 줌
- 복잡한 생태계가 자연스럽게 형성되는 과정을 경험

### 🎮 사용자 역할
| 할 수 있는 것 | 할 수 없는 것 |
|--------------|--------------|
| 관찰 및 추적 | 생명체 직접 생성 |
| 재앙 발생 | 생명체 조종 |
| 시간 조절 | 유전자 수정 |
| 환경 모니터링 | 환경 직접 변경 |

## 주요 시스템

### 1. 환경 시스템
- **물리 법칙**: 중력, 부력, 열전도
- **대기 시스템**: O₂, CO₂, N₂ 농도, 기압, 습도
- **기후 시스템**: 온도, 강수량, 계절 변화
- **수계 시스템**: 바다, 강, 호수, 습지
- **지형 시스템**: 산, 평원, 사막, 숲

### 2. 생명 시스템
- **유전자 기반 진화**: 신체, 뇌, 외형 유전자
- **AI 두뇌**: 신경망 기반 의사결정
- **프로시저럴 외형**: 유전자에 따른 무작위 외형 생성
- **성선택**: 외형 기반 짝짓기 선호도

### 3. 관찰 시스템
- **개체 추적**: 실시간 상태, AI 판단 모니터링
- **종 통계**: 개체 수, 유전 형질 변화 그래프
- **가계도**: 조상-후손 관계 시각화
- **생태계 대시보드**: 전체 통계 및 환경 상태

### 4. 재앙 시스템
- 운석 충돌, 화산 폭발, 전염병
- 빙하기, 온난화, 대홍수
- 대기 변화, 방사능 증가

## 기술 스택

- **Frontend**: React + TypeScript
- **렌더링**: WebGL/WebGPU + Canvas 2D
- **AI 엔진**: TensorFlow.js (신경망)
- **병렬 처리**: Web Workers
- **저장소**: IndexedDB

## 디자인 방향

### 비주얼 스타일
- **하이브리드 방식**: 줌 레벨에 따라 다른 표현
  - 대륙 뷰: 점/도형으로 단순화
  - 지역 뷰: 중간 수준 디테일
  - 개체 뷰: 상세한 프로시저럴 생명체
- **색상 톤**: 자연 다큐멘터리 스타일 (자연스러운 톤)

### 세계 규모
- 대륙 스케일의 광활한 맵
- 다양한 기후대와 생태계
- 수백만 개체 동시 시뮬레이션
